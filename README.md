# Alumni-ID-Card-Portal
A web-based Alumni ID Card Management System with Admin Approval, Secure Login, and ID Card Generation built using PHP, MySQL, and AdminLTE.
##  Features

###  User Panel
- Secure Login & Session Management
- Registration Submission
- Edit Details (Before Approval)
- View Registered Details
- Generate Alumni ID Card (After Approval)
- Change Password
- Application Status Tracking (Pending / Approved / Rejected)

###  Admin Panel
- View All Registered Alumni
- Approve / Reject Applications
- Add Remarks for Rejection
- Manage Records
- Secure Admin Authentication

---

##  Workflow

1. Alumni registers on the portal.
2. Admin reviews the application.
3. Application status:
   -  Approved → ID Card available
   -  Rejected → Reason displayed
   -  Pending → Under review
4. Alumni can generate and download the ID Card after approval.
